A work-in-progress Switch emulator focused on native recompilation.

Nx86 is an emulator project. The bet is simple: compile as much as possible before launch, use runtime feedback to find what was missed, then rebuild better native code.

Pipeline

AArch64 -> NxIR -> x86_64-v4

No downloads yet. It does not run Switch games yet.

Right now

  • Full compiler pipeline: decode, lift, optimize, verify, lower, package
  • Multi-block dispatcher with emergency JIT fallback
  • Block chaining and fastmem for native-speed execution
  • Self-modifying code detection, atomics v0, barrier semantics, and guest threading v0
  • Floating-point scalar and SIMD/NEON v0 in interpreter and evaluator
  • Advanced packed vector lowering, hot/cold splitting, and native coverage metrics
  • 255 passing tests across the full pipeline
  • GUI shell on Linux with Inspector and Scheduler
  • x86_64-v4 as the first target
