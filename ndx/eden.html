Drop a folder using any of these layouts and NDX picks it up, with or without a manifest:

romfs/
romfs_ext/
exefs/
cheats/

When there is no manifest, Nx86 generates an nxmod.toml from what it can determine. Fields it cannot infer: build_id in particular: are left at safe defaults. For ExeFS mods and cheats, fill in the Build ID before using it.

Nx86 does not scan existing emulator mod folders by default. Enable folder discovery in settings to point it at paths you already have.
