Profiles are named mod sets stored as TOML files, shareable and importable. Common examples: Default, Vanilla, Randomizer, HD Pack, Debug. Create, rename, and delete them freely: there is no locked Vanilla profile.

Temporary launch profiles let you run a session with a different mod set without touching your saved state. When the session ends, the previous profile is active again.

Exported profiles list mods by ID and version. The mod files themselves are not bundled: importing a profile on another machine requires those same mods to be installed there.
