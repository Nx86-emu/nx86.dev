Nx86

A work-in-progress Switch emulator focused on native recompilation.

Index About Status Legal

Status

This is still a prototype. The useful work right now is mostly inside the compiler pipeline, tests, and storage format.

Now

  • Synthetic ARM64 programs can move through the early pipeline.
  • NxIR exists as the shared middle layer.
  • There is a first tiny native block path for x86_64.
  • Local phase notes include register allocation and a first object format.

Not yet

  • No public downloads.
  • No game import.
  • No firmware, keys, or bundled system files.
  • No homebrew execution yet.

Next

  • Cache manager.
  • Dispatcher.
  • Emergency JIT path.
  • Profile-guided rebuilds.
